📖 Cara setup Google Sheets (klik untuk buka)
  1. Buka spreadsheet Anda di Google Sheets.
  2. Menu File → Share → Publish to web.
  3. Pilih sheet yang ingin dipublish (atau "Entire Document"), lalu pilih format Comma-separated values (.csv).
  4. Klik Publish. Copy URL yang muncul, paste di kolom URL di atas.
  5. Atau cara lebih simpel: cukup paste link spreadsheet biasa — pastikan akses sharing-nya minimal "Anyone with the link can view".

⚠️ Pastikan baris pertama spreadsheet adalah header (Tgl., No Dok., Kode Gudang, dst.) sesuai contoh data Anda.

📋 Kolom yang dibutuhkan (header di baris 1):

Tgl., No Dok., Kode Gudang, Kode Departemen, Kode Barang, Nama Barang, Qty, Harga, Diskon, Total (IDR), Kode Sales, Cek Kota, Week, Bulan, Brand, Cek Juta, Cek Dept, Dimensi, Cek Ink Tank.
